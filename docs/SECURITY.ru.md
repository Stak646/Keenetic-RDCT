# Безопасность и приватность (MVP)

- RDCT работает в режиме **USB-only**: если базовый путь не на внешнем носителе — запуск запрещён.
- RDCT **не отправляет** собранные данные никуда.
- WebUI/API поднимается локально. API защищён токеном (см. `config/rdct.json`).
- В режимах Light/Medium применяется **best-effort редактирование** (mask/redaction) для командных выводов и экспортов конфигов.
- `SensitiveScannerCollector` формирует редактированный отчёт по чувствительным данным и план редактирования для экспорт‑архивов.

Перед передачей отчётов третьим лицам:

1) проверьте `reports/top_findings.json`
2) проверьте `security/sensitive_findings.json`
3) используйте `rdct export --level strict` для редактированного пакета.
