# Зависимости и авто-установка

RDCT может (опционально) ставить недостающие утилиты, необходимые коллекторам.

## Реестр

RDCT ведёт локальный реестр на USB:

- `<base>/deps/registry.json`

Там фиксируется:

- что было установлено
- когда
- каким компонентом

## Offline mode

Если на роутере нет сети, RDCT переходит в **offline** режим:

- не делает auto-install
- отключает коллекторы, которым не хватает утилит

## Конфигурация

См. `docs/CONFIG_REFERENCE.md` → `dependencies` и `modes.network_policy`.
