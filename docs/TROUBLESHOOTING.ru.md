# Troubleshooting / Решение проблем

## Ошибка "USB-only enforced"

RDCT откажется запускаться, если базовый каталог не на внешнем USB.

- Проверь, что USB смонтирован:

```sh
cat /proc/mounts | grep -E '/tmp/mnt|/media|/mnt'
```

- Укажи `--base` явно:

```sh
/tmp/mnt/sda1/rdct/rdct.sh preflight
```

## Permission denied

Некоторые коллекторы требуют прав администратора/рута.

- Запускай от пользователя с правами администратора
- Переходи на `medium/full` только когда нужно

## Не хватает утилит

RDCT может (опционально) поставить зависимости через `opkg` (Entware) — зависит от конфигурации.

- Убедись, что `/opt` доступен и находится на USB
- Убедись, что есть интернет для `opkg update`

## WebUI не открывается

- Запусти сервер на правильном интерфейсе:

```sh
/tmp/mnt/sda1/rdct/rdct.sh serve --bind 0.0.0.0 --port 8080
```

- Проверь, что порт не занят.

## Отчёты слишком большие

- Используй `light`/`medium`
- По умолчанию держи mirror выключенным (`modes.mirror_policy.enabled=false`)
- Делай export со строгой редакцией (stub'ы уменьшают размер)
