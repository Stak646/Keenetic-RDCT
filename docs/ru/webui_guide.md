# Руководство WebUI

## Страницы
- **Панель управления**: Статус, быстрые действия
- **Preflight**: Проверка окружения, предупреждения
- **Прогресс**: Текущий сборщик, очередь, метрики Governor
- **Отчёты**: Список, скачивание, удаление
- **Инвентаризация**: Карта порт→процесс→пакет
- **Проверки**: Аномалии по категориям и severity
- **Цепочка**: Визуализация baseline/delta
- **Устройство**: Модель, ОС, RAM, хранилище
- **Приложения**: Статус сервисов, управление
- **Настройки**: Язык, режим

## Доступ
- URL: http://127.0.0.1:<port>
- Токен: `var/.auth_token`
- Роли: admin (полный доступ), readonly (только просмотр)
