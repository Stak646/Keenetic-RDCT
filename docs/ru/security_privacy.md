# Безопасность и приватность

## Режимы маскирования

| Режим | Поведение | Назначение |
|---|---|---|
| Light | Обнуление паролей/токенов/ключей | Безопасная передача |
| Medium | + Хэш IP/MAC, частичная маска токенов | Тикеты поддержки |
| Full | Сохранение всего, отметка находок | Глубокое расследование |
| Extreme | Сохранение без маскирования | Форензик-анализ |

## Что маскируется (Light/Medium)
- Пароли → `***REDACTED***`
- Bearer токены → `Bearer ***REDACTED***`
- Приватные ключи → `***PRIVATE KEY REDACTED***`

## Очистка для передачи
```shell
keenetic-debug sanitize <report_id>
```

## Предупреждение Full/Extreme
Режимы Full/Extreme сохраняют чувствительные данные. CLI требует флаг `--i-understand`.
