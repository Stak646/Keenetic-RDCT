Безопасность

WebUI доступен только в LAN. Используется Bearer Token аутентификация.
Роли: admin (полный доступ), readonly (только просмотр).
Опасные операции (restart/restore/delete) требуют dangerous_ops=true в конфиге.
Токен хранится в var/.auth_token (chmod 600).
